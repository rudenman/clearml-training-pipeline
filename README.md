В рамках проекта реализован полный цикл машинного обучения с использованием платформы ClearML для трекинга экспериментов.

Трекинг ML-экспериментов в ClearML
по дисциплине «Управление конфигурацией программного обеспечения»

Задача сформулирована как бинарная классификация — прогноз возврата клиента на основе агрегированных поведенческих признаков, сформированных из истории покупок.

Проект демонстрирует:

-организацию ML-экспериментов

-логирование гиперпараметров и метрик

-сравнение моделей

-воспроизводимость результатов

Настройка подключения к ClearML может выполнялась через задание переменных окружения в коде (демонстрационный вариант) в отдельном файле, который не опубликован.
